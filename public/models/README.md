# 3D Models Directory

Place your GLB model files here.

## Expected File

- `stackhouse.glb` (or your model filename)

## Usage

The model will be loaded from this directory in the portfolio application. After adding your model file, commit and push to GitHub, then provide the raw GitHub URL to update the component loading code.

## GitHub Raw URL Format

Once your file is pushed, the URL will be:
```
https://raw.githubusercontent.com/yourusername/repo-name/main/public/models/stackhouse.glb
```
